    About This Project

    ML-powered stock-direction analysis for NIFTY 50 using XGBoost + Random Forest ensemble.

    01
    Pick or Upload
    Select a NIFTY 50 stock or upload your own CSV/XLSX dataset.
    02
    Train + Predict
    XGBoost trains on engineered features, then generates a directional signal with confidence.
    03
    Explore Results
    View SHAP drivers, regime analysis, risk metrics, and backtest comparisons.
    Pipeline Details
    • Engine: XGBoost (primary) + Random Forest, weighted ensemble
    • Features: RSI, MACD, Bollinger, ATR, OBV, ADX, Stochastic, momentum, volume ratios, lagged returns
    • Target: 5-day forward return direction (bullish if >1.5%)
    • Explainability: SHAP values (fallback: native feature importance)
    • Data: Local CSV files → Yahoo Finance fallback
    Decision-support tool for education and analysis, not financial advice.
